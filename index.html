Northern Virginia Community College

Spring/Fall 2019: ITP 270 Programming for Cybersecurity (Hybrid - 8 weeks)

(Catalog:) Teaches scripting and software development techniques for automating security tasks such as network monitoring and penetration testing using Python. Additional topics include writing custom tools and the basics of developing software exploits. This course provides students with the knowledge to develop and maintain effective programs/scripts, through demonstrations and hands-on labs. Students will learn to create high-level language scripts and programs using methods, modules, classes, and other pertinent parts of the object-orientated and structured paradigms. Students will learn how to use Python to manipulate data, automate tasks, perform error handling, storing, retrieving and usefulness in a variety in cybersecurity areas. The class will provide hands on exercises to reinforce learning and develop real competency, as students are guided through the process of developing and testing security tools. To conclude the course, students will participate in a capstone project to fortify knowledge and skills gained during the course.
